# AI 代码辅助助手 (AI Code Assistant)

## 简介
这是一个基于 PySide6 (Qt) 和 WebEngine 构建的现代化 AI 代码辅助桌面应用。它结合了强大的大语言模型（支持 Qwen 和 DeepSeek）与本地文件系统读取能力，旨在为开发者提供具备深度思考能力、支持长上下文和代码库级别理解的智能对话体验。

## 核心特性
- 🧠 **深度思考模型支持**：原生支持 Qwen 和 DeepSeek 模型的推理/思考过程（Reasoning），并在 UI 中优雅折叠展示。
- 📂 **代码库级上下文**：支持加载本地文件夹，以树状图展示文件结构，自动估算 Token 消耗，精准勾选文件作为上下文。
- 💬 **多会话与长期记忆**：支持多轮对话管理、会话重命名/删除，并自动截取近期历史构建长期记忆。
- 🎨 **现代化毛玻璃 UI**：基于 HTML/CSS/JS 打造的前端界面，支持三栏拖拽布局、流式输出优化、Markdown、代码高亮及 KaTeX 数学公式渲染。
- ⚙️ **丰富的个性化设置**：提供详尽的模型参数配置，并支持自定义聊天背景壁纸。

## 前端功能与交互详解

前端采用 `QWebEngineView` 结合原生 JS/CSS 构建，通过 `QWebChannel` 与 Python 后端进行高效的双向通信。

### 1. 界面布局与交互
- **三栏自适应布局**：左侧为对话列表，中间为主聊天区，右侧为文件上下文树。
- **拖拽调整**：支持通过拖拽分隔条自由调整左侧边栏宽度、右侧边栏宽度以及底部输入区域的高度。
- **流式渲染优化**：在 AI 回复的流式输出阶段，采用纯文本追加以避免频繁 DOM 重排；输出完成后，统一进行 Markdown 解析与代码高亮渲染，保证极致流畅。

### 2. 消息与内容渲染
- **思考过程展示**：自动识别模型返回的 `reasoning_content`，以可折叠的“思考过程”面板展示，方便查看 AI 的逻辑推演。
- **多媒体与文件卡片**：用户发送的文件或代码内容，在对话流中以精美的折叠卡片形式展示，支持展开查看源码。
- **代码一键复制**：所有代码块右上角悬浮显示“复制”按钮，通过 Qt 原生剪贴板 API 绕过浏览器限制，实现一键复制。
- **重新生成**：支持对用户消息进行“重新生成”操作，自动清理后续对话并重新请求。

### 3. 文件上下文管理 (右侧边栏)
- **文件树浏览**：输入本地文件夹路径后，递归生成文件树，支持目录的展开与折叠。
- **Token 预估**：自动计算并显示每个文件的预估 Token 数和字符数，帮助用户控制上下文长度。
- **批量选择**：支持“全选”与“清空”操作，勾选的文件将作为附加上下文随消息发送给大模型。

### 4. ⚙️ 设置功能
应用提供了完善的设置面板，允许用户深度定制 AI 行为与界面表现。点击输入框旁边的设置图标（⚙️）即可打开配置窗口：

#### 模型与 API 配置
- **API Keys 管理**：独立配置 DeepSeek 和 Qwen 的 API Key。
- **模型切换**：支持在 `qwen3.7-max` 和 `deepseek-v4-pro` 等模型间无缝切换。
- **动态配置项**：
  - 当选择 **Qwen** 模型时，界面会动态显示 `Qwen Workspace ID` 配置项。
  - 当选择 **DeepSeek** 模型时，界面会动态显示 `思考强度` (low/medium/high) 配置项，以控制模型推理的深度。

#### 对话与性能参数
- **记忆轮数 (Memory Rounds)**：自定义构建长期记忆所截取的历史对话轮数（1-1000轮），平衡上下文长度与记忆力。
- **Max Tokens**：设置模型单次回复的最大 Token 限制（最高支持 1,000,000），满足长代码生成的需求。

#### 界面个性化
- **自定义壁纸**：应用支持设置自定义聊天背景壁纸，打造专属的沉浸式视觉体验。

## 后端架构与技术栈

### 技术栈
- **GUI 框架**：PySide6 (Qt6)
- **Web 渲染**：QWebEngineView (Chromium) + QWebChannel
- **前端**：原生 HTML5 / CSS3 / JavaScript (ES6+)
- **第三方前端库**：`marked.js` (Markdown), `highlight.js` (代码高亮), `KaTeX` (数学公式)
- **AI 接口**：OpenAI 兼容 API (openai-python)

### 核心模块说明
| 模块路径 | 功能描述 |
| --- | --- |
| `ui/ui.py` | 主程序入口，初始化 Qt 窗口、WebEngine 及 QWebChannel 桥接对象 (`Bridge`)。 |
| `ui/config.py` | 前端设置对话框的 Qt 实现，处理配置文件的读取与保存。 |
| `chatbot/port.py` | AI 客户端封装，处理 Qwen/DeepSeek 的流式 API 请求及思考内容解析。 |
| `chatbot/message_build.py` | 消息构建器，处理文件读取（Base64/文本）、多模态内容封装及对话记录的本地持久化。 |
| `chatbot/memory_manager.py`| 记忆管理器，按设定轮数清洗并截取历史对话，转换为 API 标准格式。 |
| `files_reader/locate.py` | 文件夹扫描器，递归生成包含文件/目录结构的树状字典。 |
| `files_reader/token_cal.py`| Token 估算器，基于文件字节数粗略估算 Token 和字符数。 |
