# YODER

基于 PySide6 + QWebEngineView 的桌面 AI 编程助手，支持多平台大模型对话、文件树管理和流式回复。

## 功能

**AI 对话**
- 支持阿里（DashScope）、DeepSeek、智谱（GLM）三个平台
- 可编辑模型下拉框，支持自定义模型名
- 流式输出 + 思考过程展示
- Markdown / KaTeX 公式 / 代码语法高亮
- 代码块一键复制

**对话管理**
- 新建 / 重命名 / 删除对话
- 删除单轮对话、重新生成回复
- 每个对话可绑定独立模型（model.json）
- 每个对话可配置记忆概括（memory.json）
- 每个对话可设置专属壁纸（wallpaper.json）

**文件支持**
- 加载文件夹，树形展示，自动忽略 `__pycache__`、`env`、`node_modules` 等
- 多文件夹管理，侧栏可收起
- 支持发送图片（base64）、PDF、Word 及文本文件
- 文件内容自动附加给 AI 并展示在前端

**记忆概括**
- 对话级配置：开关、概括轮次（默认10）、字数上限、合并段数
- 后台概括，不阻塞用户操作
- 超出短期记忆轮数或 token 超限时自动启用概括
- 重新生成不影响已有概括内容

**界面**
- 三栏布局：左侧对话列表、中间聊天区、右侧文件树
- 所有分隔条可拖拽调整
- 自定义壁纸（全局或对话级）
- 气泡宽度 70%，行间距 1.9

## 菜单说明

| 按钮 | 位置 | 功能 |
|------|------|------|
| ⚙️ | 输入框左下角 | 全局设置（平台、API Key、模型、思考强度、记忆轮数、Max Tokens） |
| 🧠 | 输入框左下角 | 记忆概括设置（开关、概括轮次、字数、合并段数） |
| ✕ | 右上角 | 关闭右侧文件栏 |
| 🔄 | 右侧路径栏 | 刷新当前文件夹 |
| ✕ | 右侧路径栏 | 移除当前文件夹 |
| + 新建对话 | 左侧底部 | 创建新对话 |
| ✎ / 🗑️ | 对话列表悬停 | 重命名 / 删除对话 |
| ✕ | 用户消息左侧 | 删除该轮对话 |
| 🔄 重新生成 | AI 回复右下角 | 重新生成该回复 |
| 全选 / 清空 | 右侧顶部 | 全选 / 取消全选文件 |
| 加载 | 文件夹路径旁 | 加载指定文件夹 |

> 空白处点击发送按钮 5 次可打开壁纸设置。

## 配置

全局配置保存在 `config/info.json`，对话级配置保存在 `records/<对话文件夹>/` 下。

## 运行

```bash
pip install -r requirements.txt
python run.py
```

## 已知缺陷

- **无搜索功能**：不支持对话内容搜索或文件树内搜索。
- **无上下文压缩**：不具备自动上下文压缩能力。对于写代码场景，通过文件树选取文件即可精确提供上下文，不需要额外的上下文压缩。
